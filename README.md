# Job-Application-Tracker

A full-stack application that helps users track and manage their job applications in one place.

Users can create, update and organise applications by status (e.g. Applied, Interview, Rejected), making it easier to stay on top of job search process.

## Tech Stack

### Frontend
- React

### Backend
- Node.js
- Express

### Database
- MongoDB

### Authentication
- JSON Web Tokens (JWT)

## Features

- User authentication (register & login)
- Create, edit and delete job applications
- Track application status
- Search and filter jobs
- Responsive user interface
